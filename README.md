# 🚀 AzaDev Portfolio

Bu mening shaxsiy portfolio loyiham bo'lib, unda bajargan IT loyihalarim, mahoratlarim va men bilan bog'lanish uchun ma'lumotlar jamlangan. Loyiha **Django** frameworkida yaratilgan.

## ✨ Xususiyatlari
* **Loyihalar galereyasi:** Bajargan ishlarim haqida batafsil ma'lumot.
* **Responsive dizayn:** Mobil va kompyuter qurilmalari uchun to'liq moslashgan.
* **Bog'lanish formasi:** Foydalanuvchilar to'g'ridan-to'g'ri xabar qoldirish imkoniyati.
* **Admin panel:** Loyihalarni osonlik bilan qo'shish va tahrirlash.

## 🛠 Texnologiyalar
* **Backend:** Python, Django 5.x
* **Frontend:** HTML5, CSS3, Tailwind CSS, JavaScript
* **Ma'lumotlar bazasi:** SQLite (mahalliy), PostgreSQL (production uchun tavsiya etiladi)
* **Virtual muhit:** venv
